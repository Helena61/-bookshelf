## table book

Where the field 'tags' is a json-field.

- id	[int]
- fk_authorid	[int]
- fk_bookshelfid	[int]
- isbn	[varchar(45)]
- title	mediumtext
- language	[varchar(45)]
- publisher	[varchar(45)]
- published_year	[varchar(4)]
- summary	[mediumtext]
- tags	[json]
- read	[enum('NOT READ','STARTED','READ')]
- onloan [tinint(1)]
